== 웹 표준

image:./images/image1.png[image1.png]

* W3C에서 권고한 표준을 지켜 웹 사이트를 제작
* 물리적 또는 인적 환경에 제약 없는 접근성
* 마우스 또는 키보드가 없는 환경
* 시각적으로 문제가 있는 상황

== 웹 호환성

image:./images/image2.png[image2.png]

* 브라우저 종류와 비전에 관계없는 접근성
* 장치의 종류와 크기에 상관없는 호환성
